def bubbleSort(alist):
for passnum in range(len(alist)-1,0,-1):
for i in range(passnum):
if alist[i]>alist[i+1]:
temp = alist[i]
alist[i] = alist[i+1]
alist[i+1] = temp

Odpowiedzi na pytania

1.Jaka jest różnica między znacznikami i oraz em? Czy zawsze dają identyczne wyniki?
Tekst wygląda tak samo. em używa się, żeby coś zaakcentować podkreślić. Emfaza jest stanem logicznym - może być używana jako kursywa, ale też na przykład jako czerwony, podkreślony tekst. i reprezentuje stan fizyczny - jest po prostu pochylonym tekstem.

2.Jaka jest różnica między znacznikami b oraz strong? Czy zawsze dają identyczne wyniki?
Znacznik strong - tekst ma znaczenie merytoryczne, jest ważny dla całości i trzeba na niego zwrócić szczególną uwagę. Znacznik b - tekst ma być pogrubiony wizualnie, ale nie jest bardziej znaczący z punktu widzenia merytorycznego. Może mieć różną formę wizualną.

3.Jaka jest różnica między znacznikami code oraz samp? Czy zawsze dają identyczne wyniki?
code stosuje się do kodów programów komputerowych, a samp do wskazywania wyników działania programu. Nie muszą dawać identycznych wyników.

4.Jaka jest różnica pomiędzy fragmentem strony sformatowanetak: <p>…</p><p>…</p> oraz fragmentem o następującej postaci: <p>…<br/><br/>…</p>?
Formatowania nie różnią się wizualnie, lecz pierwsza opcja tworzy dwa paragrafy, a druga jeden, oddzielony białymi liniami.
